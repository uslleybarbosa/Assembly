# Estudos em Assembly

Este repositório tem como objetivo reunir exemplos, exercícios e explicações sobre a linguagem **Assembly**, servindo como material de estudo para quem deseja compreender melhor a programação de baixo nível

## 🎯 Objetivo

- Explorar conceitos fundamentais da linguagem Assembly.
- Demonstrar como manipular memória, pilhas e registradores.
- Apresentar exemplos práticos.
- Servir como guia de aprendizado para curiosos.
