# Repositório Assembly

Este repositório é dedicado ao estudo e prática da linguagem **Assembly**, com foco principal em programas desenvolvidos para **arquitetura 32-bit no Linux**.  
Aqui você encontrará exemplos, exercícios e códigos que exploram desde conceitos básicos até manipulações mais avançadas em baixo nível.

## O que você vai encontrar

- **Exemplos clássicos** como [Hello World](ca://s?q=Hello_World_em_Assembly_32_bit) usando `int 0x80`.
- **Manipulação de memória** em Assembly 32-bit.
- **Operações binárias** e lógicas fundamentais.
- **Chamadas de sistema (syscalls)** no Linux 32-bit.
- Estruturação de programas simples e didáticos para compreender como o Assembly interage diretamente com o sistema operacional.

## Objetivo

O propósito deste repositório é servir como um guia prático e um acervo de códigos para quem deseja:
- Aprender Assembly do zero.
- Entender como funcionam as chamadas de sistema em 32-bit.
- Explorar a lógica de programação em baixo nível.
- Ter exemplos prontos para compilar e executar em distribuições Linux.
